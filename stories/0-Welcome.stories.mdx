# Get started

```shell
git clone https://github.com/InseeFr/IAM
cd IAM
yarn
yarn start-storybook
```

# Include IAM in my project

If you want to add the module into your own project, you should first install it via NPM or yarn

```shell
npm i @inseefr/iam bootstrap@3.4.1
```

Then, you need to include the CSS stylesheet of `bootstrap`, and another one from `@inseefr/wilco`, the components library used by **IAM**.

```javascript
import 'bootstrap/dist/css/bootstrap.css';
import '@inseefr/wilco/dist/index.css';
```

And finally, you can import and use the component we provide.

```javascript
import Habilitation from '@inseefr/iam';

...
<Habilitation
        loadRoleList={...}
        loadAgentList={...}
    />
```
